# Purch

**The UNC sublease marketplace.**  
Find your perch.

Purch connects UNC students looking to sublease their apartments with students looking for a place to stay — no broker, no Craigslist, no nonsense. Student-to-student, Chapel Hill to Chapel Hill.

---

## What it does

- Browse sublease listings near UNC's campus
- Post your sublease in minutes with photos, availability dates, and pricing
- Message listers directly in-app
- Get notified instantly via email when someone messages you
- View listings on an interactive map

---

## Tech stack

- **Frontend** — React + TypeScript + Vite + Tailwind CSS
- **Backend** — Supabase (Postgres + Auth + Storage + Realtime)
- **Email** — Resend
- **Maps** — Mapbox
- **Deployment** — Vercel
